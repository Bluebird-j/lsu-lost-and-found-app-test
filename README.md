# LSU Lost an Found - React Frontend

## Project Description
This is the React.js frontend for the LSU Lost and Found application. It allows users to report lost and found items with relevant details and contact information. Users can also sign up and log in to manage submissions.

## Core Features (MVP)
- Navigation Bar with links to Home, Lost, Found, Login, and Sign Up pages.
- Landing Page with call-to-action buttons for reporting lost or found items.
- Sign Up and Login Pages with form validation for empty fields and proper email/password format.
- Lost Page: Submit lost items with validation (name, email, item type, item name, last seen date, phone number).
- Found Page: Submit found items with validation (name, email, item type, item name, found date, phone number).
- Footer with LSU site links and contact emails.

## Libraries and Tools Used
- React.js
- React Router (`react-router-dom`)
- CSS (custom per page)
- Supabase / fetch API (placeholder for backend integration)
- useState for managing component state

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/dkdkdkd222/lsu-lost-and-found-app.git
   ```

2. Navigate to the project folder: 
   cd lostandfound
   cd frontend
   cd lost-and-found

3. Install any dependencies
   npm install

4. Start the development server
   In the directory lost-and-found run:
   - npm start
